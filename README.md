# ETH-Tx-Overview-2025
Built an interactive dashboard analyzing Ethereum transaction activity in 2025, including transaction volume, gas fees, active wallets, and network trends.



On-chain analytics dashboard built on Dune Analytics to analyze Ethereum transaction activity in 2025.

## What this covers
- Ethereum transaction volume and count
- Gas fee trends
- Network activity over time
- User and transaction behavior insights

## Tools
- Dune Analytics
- SQL
- Ethereum on-chain data
